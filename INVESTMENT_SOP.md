# 投资定投实操手册 (SOP)

## 1. 核心资产配置 (20,000元/月)

基于“现金流覆盖”与“本金安全”的目标，采用 **2 : 4 : 3 : 1** 的动态分配模型。

| 板块 | 比例 | 金额 | 标的建议 | 角色 |
| :--- | :--- | :--- | :--- | :--- |
| **增长层** | 10% | 2,000 | 纳指 (QQQM) / 比特币 (IBIT) | **彩票/高赔率** (雷打不动，定投) |
| **防御层** | 20% | 4,000 | 美元短债 (SGOV) / 存款 | **弹药库** (平时吃息，战时补仓) |
| **A股红利** | 40% | 8,000 | 红利低波 (510880/563020) | **主力现金流** (根据脚本信号操作) |
| **港股红利** | 30% | 6,000 | 建行 (0939) / 中海油 (0883) | **高息现金流** (根据脚本信号操作) |

## 2. 每月发薪日操作流程

**时间:** 每月发薪日 (e.g. 5号)
**工具:** 终端 (Terminal)

### Step 1: 盲投 (耗时 1分钟)
*   不管市场涨跌，直接买入 **2,000元** 的增长层资产 (QQQM/BTC)。
*   *心态: 这一步是为了保持对高增长资产的敞口，不要择时。*

### Step 2: 获取信号 (耗时 30秒)
在终端运行脚本：
```bash
./run.sh
```

### Step 3: 资金路由 (关键决策)
根据 `src/config.py` 的计算结果，决定剩余 **18,000元** (A股8k + 港股6k + 现金4k) 的去向。

#### 信号对照表

| 脚本信号 (颜色) | 含义 | 操作指令 | 资金流向 |
| :--- | :--- | :--- | :--- |
| **🔵 [BUY] / 🟡 [HOLD]** | **正常/观望** | **按部就班** | **A股买8k, 港股买6k, 存美元4k**。<br>*(维持默认配置，积攒子弹)* |
| **🟢 [STRONG BUY]** | **机会/低估** | **吸血模式** | **该板块预算 + 挪用当月现金(4k) + 动用存量储备**。<br>*(例如港股Strong Buy: 买入 6k+4k=10k 港股，甚至更多)* |
| **🟠 [STOP]** | **过热/停买** | **熔断模式** | **该板块预算归零 -> 全部转入美元短债(SGOV)**。<br>*(例如A股Stop: 8k不买了，直接买8k SGOV)* |
| **🔴 [SELL]** | **极贵/止盈** | **收割模式** | **停买 + 卖出部分持仓 -> 全部转入美元短债(SGOV)**。<br>*(落袋为安，等待回调)* |

## 3. 场景演示

### 场景 A: 也就是那样 (平淡)
*   **信号:** A股 `HOLD`, 港股 `BUY`.
*   **操作:**
    *   2k -> QQQM (雷打不动)
    *   4k -> SGOV (存起来)
    *   8k -> A股红利ETF
    *   6k -> 港股建行

### 场景 B: A股疯牛 (高估), 港股遍地黄金
*   **信号:** A股 `STOP` (收益率 < 商贷), 港股 `STRONG BUY` (利差极高).
*   **操作:**
    *   2k -> QQQM
    *   A股的 8k -> **不买A股**，全部转移。
    *   资金池 -> 4k (现金) + 8k (A股转移) + 6k (港股预算) = **18,000元**。
    *   **All in 港股**: 用 18,000 元买入港股红利标的。

### 场景 C: 全球股灾 (暴跌)
*   **信号:** A股 `STRONG BUY`, 港股 `STRONG BUY`.
*   **操作:**
    *   2k -> QQQM
    *   **动用战略储备**: 卖出之前存的 SGOV (假设已存了5万)。
    *   **本月投入**: 18,000 (工资) + 50,000 (存款) = **68,000元**。
    *   疯狂扫货廉价的红利资产。

## 4. 为什么这么做？
1.  **跑赢房贷:** 只要脚本不让你买（STOP），你的钱就去了 SGOV (4%+) 或还贷，这天然跑赢了商贷利率。
2.  **避免接盘:** 脚本阻止了你在高位 (STOP) 投入本金，保护了现金流的源头。
3.  **敢于抄底:** 只有平时攒下了“防御层”的钱，在 `STRONG BUY` 出现时你才有钱敢买，从而锁定极高的股息率。
